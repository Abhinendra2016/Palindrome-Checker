# Palindrome Checker

This is a simple web application that checks whether a given word or phrase is a palindrome. A palindrome is a word or phrase that reads the same backward as it does forward.

## Features

* Checks whether a given word or phrase is a palindrome
* Displays a message indicating whether the input is a palindrome or not

## Technologies Used

* HTML
* CSS
* JavaScript

## Installation

1. Clone the repository to your local machine
2. Open the index.html file in a web browser

## Usage

1. Enter a word or phrase in the input field
2. Click the "Check Palindrome" button
3. The result will be displayed in the "Result" div
